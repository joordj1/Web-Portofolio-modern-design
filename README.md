# ✨ Futuristic Portfolio Web

Website portfolio modern dengan tampilan futuristik, animasi interaktif, custom sparkle cursor, smooth scrolling, dan responsive layout. Dibangun menggunakan teknologi modern untuk memberikan pengalaman visual yang premium sekaligus performant. Karena apparently manusia sekarang tidak cukup hanya menampilkan CV biasa. Harus ada glow, particle, dan transisi kosmik agar dianggap kompeten.

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

## 2. Masuk ke Folder Project

```bash
cd your-repository
```

## 3. Install Dependencies

Menggunakan npm:

```bash
npm install
```

Atau pnpm:

```bash
pnpm install
```

Atau yarn:

```bash
yarn install
```

---

# ▶️ Running Development Server

Menggunakan npm:

```bash
npm run dev
```

Atau pnpm:

```bash
pnpm dev
```

Atau yarn:

```bash
yarn dev
```

Buka browser:

```bash
http://localhost:3000
```

Jika ingin diakses melalui jaringan lokal:

```bash
http://192.168.x.x:3000
```

Karena tentu saja setelah localhost berhasil, langkah berikutnya adalah menemukan bug baru saat dibuka di perangkat lain. Tradisi web development yang tak pernah mati.

---

# 🏗️ Build Production

```bash
npm run build
```

Menjalankan production server:

```bash
npm start
```
